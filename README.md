# YPracticum
Здесь вкратце можно ознакомиться с задачами и результатами, полученными в процессе решения учебных проектов Яндекс Практикум
### Теледом. Прогноз оттока клиентов
**Бизнес-задача:** избежать потери прибыли в результате оттока клиентов

**Решение:** построить и внедрить модель, предсказывающую расторжение договора клиентом

**Результат:** модель на тестовых данных предсказывает будущее поведение клиентов с точностью в 92% (ROC-AUC)

Подробную постановку задачи, проведенный исследовательский анализ данных, использованные модели МО, вывод и рекомендации можно посмотреть в [ноутбуке](telecom.ipynb)


### Проект для «Викишоп»
**Бизнес-задача:** разработать инструмент для оперативного выявления токсичных комментариев

**Решение:** разработать модель способную выявлять токсичные комментарии на основании инструментов обработки естественного языка 

**Результат:** модель на тестовых данных показыват результат по метрики качества F1=0,766 

Подробную постановку задачи, проведенный исследовательский анализ данных, использованные модели МО, вывод и рекомендации можно посмотреть в [ноутбуке](NLP.ipynb)
